# Sistema de BINGO

Programa simples de bingo em Node.js criado pela EQUIPE 3 da Turma 130 do Programa Transforme-se (SENAC PE)

## Descrição

O sistema permite iniciar o jogo, apresentar um menu de opções e encerrar o programa. O código principal está em `sistema_de_bingo.js`.

## Requisitos

- Node.js instalado
- Pacote `readline-sync`

## Instalação

1. Abra o terminal.
2. Navegue até a pasta do projeto.
3. Execute `npm install readline-sync`.

## Uso

No terminal, execute:

```
node "c:\Users\shiny\Documents\Sistema de BINGO\sistema_de_bingo.js"
```

## Funcionalidades

- Inicia o sistema e solicita o nome do usuário.
- Exibe menu com opções de bingo.
- Permite encerrar o programa.
